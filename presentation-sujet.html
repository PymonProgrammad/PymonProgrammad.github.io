Principe :

Il s'agit d'une variante du jeu des n-reines. Chaque case reçoit un nombre de points et chaque reine rapporte le nombre de points de la case où elle a été placée. Des points sont retirés pour chaque paire de reines pouvant se prendre (étant donc en conflit).

Le but est d'obtenir le meilleur score possible selon ces règles.


Données :

  • Un échiquier carré de taille n avec un nombre de points sur chaque case
  • Un nombre de jetons à placer x
  • Une valeur de pénalité p pour chaque conflits

Exemple :

Calcul du score :

Le score se calcule de la façon suivante :
score = score brut - nb de conflit * pénalité
le score brut est la somme de tous les points des cases sur lequelles les reines sont placées

Exemple :

Sur cet exemple, on a un score brut = 9 + 8 + 2 + 3 = 22.
On a 3 conflits sur les colonnes et 2 conflits sur les diagonales.
On a donc 5 conflits. La pénalité sur cet exemple est de 10.
On aura donc comme score : 22 - 5*10 = -28.

Communication

Afin de faciliter l'échange d'informations sur une grille ou sur une solution, des formats de fichier standards ont été mis au point.

Fichier d'instance

Une instance est constituée des informations de taille, de nombre de jetons et de pénalité suivies des valeurs de la grille disposées en matrice n×n.

n x p
v11 v12 v13 ...
v21 v22 v23 ...
v31 v32 v33 ...
...

Fichier solution

Une solution est constituée des numéros des cases où se trouvent les jetons (de 1 à n²) puis du score de cette solution.

j1 j2 j3 ...
score

Vous pouvez charger ce type de fichiers sur la page d'accueil pour tester une instance de votre choix.
Il est également possible de télécharger les instances et solutions que vous trouvez intéressantes.
