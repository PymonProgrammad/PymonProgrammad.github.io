Problème de l'échiquier

Le Problème

Présentation du sujet

Présentation du groupe

Développement de l'application

Principe :

Il s'agit d'une variante du jeu des n-reines. Chaque case reçoit un nombre de points et chaque reine rapporte le nombre de points de la case où elle a été placée. Des points sont retirés pour chaque paire de reines pouvant se prendre.

Le but est d'obtenir le meilleur score possible selon ces règles.

Données :

  • Un échiquier de taille n (n = nombre de ligne (idem pour colonne))
  • Un nombre de jetons x à placer
  • Une valeur de pénalité p pour chaque conflits
